Game Server

project

Car combat was designed and built for CS481(Senior Software Project). Car combat is a real-time multiplayer game. Car combat uses an architecture with an authoritative server. Each client connects to the server and communicates over a UDP socket. Each client streams commands to the server, and the server broadcasts the current State of the Game to all of the connected clients.
